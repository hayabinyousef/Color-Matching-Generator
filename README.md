# Color-Matching-Generator
## Overview 
this project is C# win form app that generates outfit color palette based on warm, cool and neutral color styles using RGB based logic
## Features 
- Generate a main color based on selected style
- Generate matching colors
- Display RGB values
- Save and remove palettes locally
## Technologies used
- C#
- .NET
- WINDOWS FORM
